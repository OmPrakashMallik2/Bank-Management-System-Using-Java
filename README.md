# Bank Management System

## Introduction
This Java-based Bank Management System allows users to create new accounts, log in, perform transactions, and manage their banking activities.

## Features
- Create a new account with personalized details.
- Log in to an existing account securely.
- Perform transactions, including cash withdrawal and credit.
- View account details and transaction statements.
- Forgot Pin feature for account recovery.

## Technologies Used
- Java
- MySQL (Database)

## How to Use
1. Run the Java program.
2. Choose options to create a new account, log in, or recover a forgotten pin.
3. Follow the on-screen instructions for each operation.

## Database Configuration
- Database URL: jdbc:mysql://localhost:3306/bank_db
- Username: root
- Password: java@12345

## Requirements
- Java Development Kit (JDK)
- MySQL Database

## Note
- Make sure to configure the database details in the code before running the program.
